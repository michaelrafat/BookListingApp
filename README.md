# Book Listing App
#### Book Listing Android App. Android Basics NanoDegree by Google on Udacity.

# Project Overview :

#### - Fetching data from an API.

#### - Using an AsyncTask.

#### - Parsing a JSON response.

#### - Creating a list based on a Search Query which user will enter to build the URL and displaying it to the user.
